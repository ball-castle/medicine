# 两本书的核心知识点与图表清单

这份清单是给产品设计和内容制作直接使用的，不再按 PDF 页码整理，而是按学习模块整理。

## 一、推荐先做的 6 个模块

1. 圆运动基础
2. 方药与经方逻辑
3. 温病与儿科本气
4. 脉法与舌象判断
5. 扶阳与坎离水火
6. 医案与活解

## 二、25 个核心知识点

### 模块 1：圆运动基础

1. 圆运动：人体是一个持续升降浮沉的整体系统。
2. 中气为轴：脾胃中气是全局承接和转运中心。
3. 升降出入：先看方向失衡，再看症状外观。
4. 五脏是功能势能：五脏更像五种力，而不是五块器官。
5. 先看病机，不先看病名：同病名可以有完全不同的结构。

### 模块 2：方药与经方逻辑

6. 方剂是气机模型：一张方代表一种调整路线。
7. 六经是整体框架：六经是分层地图，不是零散条文。
8. 药性看运动方向：先问药把气往哪儿带。
9. 误治会改写病机：错误处理会制造新的复杂层次。

### 模块 3：温病与儿科本气

10. 温病先看本气：热象背后可能是中气和津液先散。
11. 儿科先护中气：小儿重在护根气，不宜猛攻。
12. 时病同名不同路数：同为暑、湿、泻，内部结构可以完全不同。

### 模块 4：脉法与舌象判断

13. 脉法先抓总开关：先判断整体大方向。
14. 舌象是辅助窗口：舌要和脉一起看。
15. 脉比病名更重要：主轴判断优先于疾病标签。

### 模块 5：扶阳与坎离水火

16. 医易互通：由局部推整体，是火神门的表达语言。
17. 坎离水火：扶阳核心在上下交通和真阳归根。
18. 扶阳不等于猛热：关键是护根、通路、归藏。
19. 桂枝法是开门拨机：让内外重新接轨。
20. 附子法是归根回阳：处理根气欲脱和阳不归根。
21. 茯神法是右降缓转：在强法不宜时先柔和转机。
22. 厥阴是转换关口：复杂逆冲和木气错乱常出在这里。
23. 上热下寒、内外不通：许多复杂表现共用这条主轴。

### 模块 6：医案与活解

24. 治疗要有先后手和节奏：不是一张方走到底。
25. 活解不是背死方：固定主轴下做灵活调度。
