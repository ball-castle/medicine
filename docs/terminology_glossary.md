# 统一术语表

这份术语表的目标不是做学术定义，而是统一：

1. 内容编写时怎么用词
2. 页面里怎么向大众解释
3. 哪些词最容易被误解

## 核心术语

| 术语 | 易懂说法 | 在产品里怎么讲 | 常见误解 |
| --- | --- | --- | --- |
| 圆运动 | 一个整体持续流动的系统 | 用总图、流线、四时变化讲“整体在转” | 误以为是抽象哲学，不是具体病机语言 |
| 中气 | 系统的承接中轴 | 用“轴心”和“转盘中心”比喻 | 误解成只等于脾胃器官本身 |
| 升降出入 | 气机运动方向 | 用“该升不升、该降不降”讲判断入口 | 误解成背术语，而不是看方向 |
| 本气 | 人体自身的根本状态 | 用“系统底盘”和“承接能力”解释 | 误解成某种神秘能量 |
| 五脏功能势能 | 五种功能方向 | 把肝、肺、脾、肾、心讲成五种力 | 误解成现代解剖器官对照课 |
| 六经 | 一张分层地图 | 强调层次、空间、位置和路径 | 误解成零散条文标题 |
| 方剂是气机模型 | 一张方就是一条调整路线 | 用“把哪股气往哪边带”来讲 | 误解成一方只对一个病名 |
| 药性方向 | 药味的运动倾向 | 先讲升、降、收、散、温、润 | 误解成药理作用列表 |
| 脉法 | 抓整体主轴的入口 | 用“先看总开关”来讲 | 误解成玄学或神秘读心术 |
| 舌象 | 辅助窗口 | 用“窗户”和“投影”来讲 | 误解成单靠看图就能诊断 |
| 扶阳 | 护住真阳和归根能力 | 强调“有根、有路、能藏” | 误解成一见寒就猛上热药 |
| 坎离水火 | 上下交通关系 | 讲“上下能不能接通” | 误解成纯易经符号游戏 |
| 桂枝法 | 开门拨机 | 讲“门轴卡住，要先拨开” | 误解成普通发汗法 |
| 附子法 | 归根回阳 | 讲“把快散掉的火收回炉膛” | 误解成单纯升温、补热 |
| 茯神法 | 右降缓转 | 讲“先把乱象缓缓带下来” | 误解成作用弱、只是安神 |
| 开门 | 让内外重新接轨 | 讲交通恢复、门轴打开 | 误解成所有扶阳都叫开门 |
| 归根 | 把浮散之阳收回根部 | 讲“守住根” | 误解成和开门是同一种动作 |
| 右降缓转 | 先稳局面再决定后续 | 讲“先别猛推，先让系统能接住” | 误解成拖延或保守 |
| 厥阴 | 转换关口 | 讲“最容易出现复杂逆冲的枢纽” | 误解成只是单条经络名称 |
| 上热下寒 | 表象和根本不一致 | 讲“上面很亮，下面接不住” | 误解成只要看到热就该清 |
| 先后手 | 治疗顺序感 | 讲“不是做不做，而是先做哪一步” | 误解成固定模板 |
| 活解 | 在主轴不变下灵活调度 | 讲“总纲不变，招式可变” | 误解成随便加减、想怎么改就怎么改 |

## 推荐统一表达

后面写页面和课程时，尽量统一用下面这些表达：

1. 少说“功效”，多说“动作”或“方向”
2. 少说“某病用某方”，多说“某种结构更适合哪条路线”
3. 少说“补热”，多说“开门、归根、缓转、交通恢复”
4. 少说“看病名”，多说“先抓主轴”

## 页面文案约束

面向大众时，优先用这些句式：

1. “先看哪里不通，不先背名字”
2. “先分清是没路、没根，还是太乱”
3. “这一步是在开门，不是在加热”
4. “这一步是在归根，不是在往外推”
5. “这一步是在先稳住局面”

## 不建议反复直接裸用的词

这些词如果直接裸用，大众会更容易误会：

1. 少阴真阳
2. 坎中一阳
3. 乾坤大挪移
4. 相火不降
5. 龙雷之火

不是不能用，而是应该先翻译成动作或结构，再补原词。
