# 第一轮重点图表分镜

这份文档对应第一批最值得进入制作的 3 张图，目的是把“讲什么”提前说清楚，避免一上来就陷入纯视觉讨论。

## 1. 圆运动总图

目标：

- 让用户第一次感到中医在看整体流动，而不是病名列表。
- 在 90 秒内建立“中气 + 升降出入 + 五脏势能”的共同框架。

关键场景：

1. 一整个系统缓慢运转
2. 四向流线分化并高亮
3. 中轴偏斜带动外围失衡
4. 收束为几个核心关键词

最关键交互：

- 时间轴切换四时
- 中轴强弱滑杆
- 五脏节点高亮

## 2. 坎离水火交通图

目标：

- 让用户不再把扶阳理解成“补热药”。
- 让用户明白关键在上下交通和归根。

关键场景：

1. 正常交通
2. 交通中断，上热下寒
3. 通路恢复，系统稳定
4. 顺势引出桂枝法和附子法

最关键交互：

- 正常/阻断状态切换
- 交通恢复进度条
- 表象与根本的双层说明

## 3. 桂枝法开门动画

目标：

- 让用户一眼理解桂枝法的动作逻辑是“开门拨机”。
- 为后续附子法做强对照。

关键场景：

1. 门轴卡住，内外不通
2. 桂枝拨动门轴
3. 流线重新贯通
4. 和附子法形成对照

最关键交互：

- 门把拖动
- 阈值解锁反馈
- 流线点选说明

## 当前建议

如果团队资源有限，制作顺序建议是：

1. 圆运动总图
2. 桂枝法开门动画
3. 坎离水火交通图

原因：

- 圆运动总图负责建立世界观
- 桂枝法动画最容易快速做出“用户一看就懂”的效果
- 坎离水火交通图价值最高，但表达门槛也最高，适合在前两张跑顺后精做
