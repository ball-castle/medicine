湿家，其人但头计出，背强，欲得被覆向火。若下之早，则
得水而不能饮，则口燥烦也。

( 吴力脂彰之脂，寒字作效字解。下有部而胸有痰，所以香上如脂也。)

病者一身画疼，发热。日晴所剧者，此名风湿。此病伤於汗出
当风，或久伤取冷所臻也。 ( 日晴，乃利丁时，此时空气收敛，风混中内区
齐。)

问上，风湿相搏，一身画疼痛，法当汗出而解。值天险两不
止，医云此可发汗，汗之病不愈者，何也? 答: 发其计，汗大出
者，但风气去，瀑气在，是故不愈也。若治风湿者，发其和汗，但微
微似欲汗出者，风湿俱去也。 (微微似葵汗出者，惟病人自己知道。)

伤寒八九日，风湿相搏，身体烦痛，不能自转侧，不电不兆，
有版浮虚而湾者，桂枝附子渴主之。若其人大便硬，小便自利者，桂
枝附子去桂加白术汤主之。 (水便利，大便硬，津液伤，洒不去。必小便减，大
便和，滔也去出。 )

风湿相捕，骨季烦 (次) 岂痛，不得屈伸，近之则痛剧。汗出短
气，小便不利，恶风不欲去胡，或身微胜者，甘草附子汤主之。 ( 汉
流因条，阳上唐不能外达。 )

瀑家病，身上疼痛，发热面黄而跨，椒痛鼻塞而烦，其有版大，
自能饮食，上腹中和然病，病在头中，寒湿故鼻塞，内药遇中则愈。
( 内查振中，药方缺。 )

瀑家下之，额上汗出，微喘，小便利者死。若下利不止者，亦
死。

(汗喘足亡藉上,便利阳亡於下，上下腊，中毛亡，故死也。以上恩章，论肖病。 )

已，泥瀑然毛和从，手足逆冷，小有劳身即热，口开前板站燥。若发
