疑难篇读法

钼伤寒论，要一眼将整个看个了然。偶因一章，疑杂费解，便
将整个耽撞。本篇读法，为能一眼了然整个之故。将疑难费解各
章，列羽最后一篇。妊人了然整个之和后，再读疑杂各章，疑难者，
亦不疑鸡侨。

类伤寒病篇
类伤寒各章

太阳病，发热而渴，不恶寒者，为温病。若发汗已，身灼热
者，名目风漫。风漫为病，且险阳俱浮，自汗出，身重，多眼睡，
奥息必冉，语言难出。若被下者，小便不利，直视，失省。若被火
者，微发黄色，出则如蓄痫，时瘾疯。若火更之，一道尚引日，再
逆促命期。

( 温妨木气跪浊之病，风乃木气琉澳之气，刘病已发汗，发计则下澳又璇澳矣。风温
云者，下泪又芍油之病也。自汗出以下诸证，局琉油之甚，肺除复亡之现象。此风字，非
风寒立风也。此一章，论温病。温病未立方，原理即方也。 )

太阳病，发热，且沉而细者，名目冶。 (津液售，故服细。 )

太阳病，发汗太多，因致冶。 ( 发于太多，故津液伤。)

病，身热足寒，杜项强急，恶寒，时头热面赤，目有版未，独标
的。浴口喉，背反张者，竣病也。 ( 身热足寒等等，交病靠等，肖津液伤所
致。痢病现证如此。)

太阳病，发热汗出，不恶寒者，名日柔疗。 ( 竣病方详金区。)

太阳病，发热悉汗，反恶寒者，名目刚疗。 (以上五章论剖病。)

泾家之为病。一身画痛，发热，身色如藏黄也。 ( 土色为黄，土所
为漂。故肖病则身黄。泽阻某征，故身疼发热。 )

太阳病，关和节疼痛而烦，有版沉而细者，此名淫丫。淫丫之候，
其人小便不利，大便反快，但当利其小便。 ( 关节疼烦，服沉而细，泽售
津，故疾痛服细。 )
