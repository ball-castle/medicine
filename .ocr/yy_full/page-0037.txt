僧寒论睛经原文请法篇

伤寒六七日，且微，手足厥冷，烦躁，灸厥险，威不逮者死。
(十日当阳气来复之期，展不还，阳不复也。)

下利手足厥冷，扰且者，灸之不温，若有版不还反微喘者，死。
( 中氧消减，故见微增。)

下利和后有版缀，手足厥冷，上时版时，手足温者生，有版不还者
死。 (上时一遇时也。)

伤寒，下利日十余行，脾反实者死。 (下利县当微弱，阳亡不能运化则
服宝。以上盛章，论磋除也亡死症。)

伤寒五六日，不结胸，腹淆有片虚复厥者，不可下。此为亡血，
下之死。 (腹汕蚤中让血塞，故下之即死。)

伤寒有还，六七日而反与黄芬泥徽其热。有版还羽蹇，今与黄芬
汤复除其热，腹中应冷，当不能食。今反能食，此名除中，必死。
( 中氧将亡民能舱者，胃氨动也。动则散人妆。以上二章，论展除死症系误谎医药者。 )

下利，有版沉纺者，下重也，孚大者为未止，有版微弱数者为欲自
止，难发热不死。 (发热不兼下利服躁者，此发熟为阳复。此一章，论厥除阳槛不
死证。 )

下利有版沉而过，其人面少赤，身有微热，下利清敲者，必瑜周
汗出而解，病人必微厥。所以然者，其面戴阳，下虚故也。 (而志向
热，阳氧上感，下利清谷，阳气下虚，汗出则上下种平，故微威病解。 )

下利有版数有微热，汗出令自愈。设复紧，为未解。 ( 听数得汗，阳
氧通调，腑复紧，阳仍未通也。以上三章，论原除碘病阳槛病解证。)

阳明胃腑病

问目: 阳明病外证云何? 答日: 身热汗自出，不要寒反恶热
也。 ( 计自出反是热，骨家阳宝之现象。 )

问上: 病有得之一日，不恶热而恶寒者，何也? 答百: 四得之
一日，恶寒将自震，即自计出而恶热也。 (冒家阳实，故恶塞之表证易
需。)

问上: 恶寒何故自因? 答上: 阳明居中土也，草物所用，息所
复传。始办恶寒，二日自止，此为阳明病也。 ( 阳强病冒阳实。乃霄家自
病，经文传字。人备意甚多。详传经篇。)
