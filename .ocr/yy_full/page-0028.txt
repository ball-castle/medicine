中风发热，六七日不解而烦，有表于证，渴答饮水，水入则吐
者，名目水道，五苓散主之。 (热芒表认，温局误证，此热乃阳马水格，非形病

太阳病，小便利者，以饮水多，必心下慰，小便少者，必苦于
急也。 (水格刚心气不降，故。)

伤寒，汗出而渴者，五苓散主之。不渴者，人茯苓甘草渴主之。
( 温击汗出融宫温盛。不温而于出矶表隔虚。以上三章，论楷衔病解腑氧之温副。 )

伤寒，胀滑而邱者，庄有热也，白虎渴主之。 ( 爆热灼津，淖流漳
腾，则服滑。内葡歼阻除氧於外，则外厥。此滑服重近有力。厥者，肢冷要寒也。 )

伤寒，脾浮滑，此表有热训有寒也，白虎汤主之。 (表热误罕，扰

伤寒，脾浮，发热，其表不解者，不可与白虎泌。渴位饮水，
狼表证者，白虎加人参渴主之。 (有玫热则衷阳虚，故不可用白虎以玫圳阳，

病人身大热，反位得近衣者，热在皮唐，寒在骨茵也。病人身
大寒，反不伐近衣者，寒在皮庚，热在骨艇也。 ( 北诊断内热之一法，不
可鬼执。以上五章，府此生病解脑气之燥动。 )
太险脾肛病

太队之为病，腹满而吐，食不下，自利益其，时腹白痛。若下
之，必胸下结硬。 (抑称太除病，皆太除有牌脾病，乃实病，非经病。少除接除准
此。此一章，论太除病之提网。除诬病寒，本苯原来除寒故也。少除厥除准此。)

自利不渴者，属太险，以其胶有寒也，宣服四逆素。 (不光三字，
芳险寒用热药之据。 )
