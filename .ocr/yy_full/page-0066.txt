太阳中蝎，身热疼重，而有版微弱，此亦夏月伤冷水，水行皮中
所致也。 (最天浴于冷水，水氧将热气团住，故发热身疼重也。以上三章，论蝎病。)

总，病有霍乱者何。答四，鹃吐而利名罩霍杞。 ( 埠者大也,又
散祥速也。升降倒行，中所将亡之大天也。 )

问斩, 病发热头痛身疼恶寒吐利者，此局何病? 答百: 此名才
乱。自吐下，利止复更发热也。 ( 兹生根于牌胃，故吐利则作寒熟。吐则伤
津，故利止复更发康。)

霍乱，枯疼发热，身疹痛，热多欲饮水者，五苓散主之。寒多
不用水者，理中丸主之。 (霍乱病，有热才顺，有寒堆乱，泾霍山，前霍天 ，寒
热混合霍弄。经文只前汉罕二种也。)

吐利汗出，发热恶寒，四肢拘急，手足厥冷者，四首汤主之。
( 寒霍于中，常有此病。阳亡标速，赤用四逆泛。)

既吐且利，小便复利而大汗出，下利清玉，内寒外热，且微欲
绝者，四道汤主之。 ( 铬利而尿又利，又大汗出，服又欲缀，阳将亡也，故用四着
滥同阳。)

吐下已断，汗出而邱，四上肢拘急不解，版微欲绝者，通有版四道
加猪胆汁渴主之。 (省出而殿，阳将亡峰，克用通县四逆同阳，加政史汁养胃路之
路，以收阳和氧也。 )

恶寒，版微而复和，利止亡血也。四逆加和人参汤主之。 (版和而
惑寒为亡血者，阳有气距微，除血亦弱也。故用四逆补阳，人参襟氧以生血。和字不可误利
字。)

叶利止而身痛不体者，当消息和解其外，宜桂枢汤小和之。 (生
冶不休为有表证，故用桂枝温。)

吐利发汗，有版平小烦者，以新虚不腾融气故也。 (服平，此病已合
之服。以上无壹论需乱。 )

伤寒，其版微汪者，本是霍乱，今是伤寒，御四五日，至险经
上。转入险必利。本吓下利者，不可治也。欲似大便而反矢气，仍
不利者，此属阳明也，便必硬，十三日愈。所以然者，经画故也。
( 未蜡下利，此是堆珊，不可用伤寒三除之法为治。便硬矢气，此是阳明，又不可用霍乱
之法移治。 )
