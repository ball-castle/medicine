阳明胃腑病

阳明之为病，胃家实也。 (一部伤寒论，惟阳明骨及有可下实刘。此一章，
论阳明胃及病之提网。 )

伤寒三日，阳明且大。 (大考，实大也。大服有康实之分。三日详传经

太阳病，三日，发汗不解，蒸燕发热者，属胃也，调胃承气渴
主之。 (证仅闵东发热，乃下家突之渐也。以上二章，论阳明再腑病成之澳。)

二阳侍病，太阳证震，但发潮热，反不能食者，胃中必有燥屎
五六枚也，宜大承气汤下之。若能食者，但硬耳。 (燥尼乃青家宝之物，
故下燥果，病力能念。柴街与阳明届腑都病称二阳供病。但硬言不燥也。 )

病人不大便五六上日，绕脐痛，烦煤，发作有时者，此有燥尿，
故使不大便也。 ( 愤中食物，被煤氧炼乾，帮笑焊尿。 )

大下和后，六七日不大便，烦不解，腹满痛者，此有燥屎也。记
以然者，有凤食故也，宜大承气泥。宿食为燥气精喜成燥尿。)

病人小便不利，大便乍难年易，时有微热，喘冒不得区者，有
燥屎也，宜大承气渴。 (小便不利，喘氧不队，皆是灯热借津。阳明下证，需小便
利，燥热伤津，故不利也。以上四章，论险明胃及下证之实据。 )

阳明病，潮热，大便微硬者，可与大承气汤，不硬者，不可与
之。若不大便六七日，巩有燥屎，少与小承气汤，渴入腹中，转矢
毛者， 此有燥屎， 万可攻之。 若不转矢气， 此但初头硬， 和后必注，
攻之必服满不能食也。答饮水者，与水则喊，其和合发热者，必大便
复硬而少也。以小承气汤和之。不转尿所者，惯不可攻也。 ( 必徘湖
热之便硬，乃可用大承氧汤下之。矢，古应字，转矢氧者，放尾也。此一章，示人蛋重用
下之法。 )
