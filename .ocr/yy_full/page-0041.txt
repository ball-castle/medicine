太阳与阳明合病，喘而胸满者，不可下，朵黄汤主之。 ( 有和荣衔
之蚂寒，有阳明之服大，百仓病。)

阳明病，且浮无汗而跨者，发汗则愈，宜麻黄汤。 (此竟奥上章均
重在民字，故主麻黄，喘融肺实。阳明之喘，肺气燥宝。内光之喘，多肺气虚。 )

阳明病，脾运，汗出多，微恶寒者，表未解也，可发汗，宜桂
枝泥。。 ( 名有缓象，言不数也。以上七章，论柴隐与隔明朝腑经气同病治法。)

太阳病，外证未解者，不可下也，下之发着。欲解外者，桂枝
泥主之。 (苏证未解而下之，兹生内陷突，故称珊地。 )

夫病有版浮大，癌病者言，但便硬耳。设利之为大关。硬为实，
汗出而解。何以故，有眠浮当以汗解。 ( 碑浮再表诊，服大为脐诈，脑证兼表
轿，当先解表，和与表证兼暴利，党先光肛，羽对待理法。)

伤寒，不大便六七日，头痛有热者，与承气汤。其小便清者，
知不在应仍在表也，当须发汗; 若头痛者必岂，宜桂枝汤。 ( 项疼有
然，阳明不降，故电。此头痛乃额角痛，脆经上逆故痛。 )

二阳普病，太阳初得病时，发其和计，汗先出不澈，因转属阳
明。续自微诗出，不亚寒，若太阳病证不密者，不可下，下之为
逆。如此，可小发其和汗。设面色缘缘正赤者，阳气拂沧在表，当解
之更之。若发汗不激，不足言，阳气拂殉不得越，当汗不和汗，其人
烦燥，不知痛处，竺在腹中，竺在四肢，按之不可得，其人短氛，
但坐以汗出不澈故也。更发汗则愈，何以知汗出不澈? 以版渍故知
也。 (降职病如柴生，先温后表，否则某生内陷。阳腑病刀荣本，先表后下，否则某生
内路，汗澈划服象和紫衔调，渚则不和不调也。 )

病人烦热，汗出则解，又如猴状，日晴时发热者，属阳明也。
脾实者，宜下之，有版浮虚者，宜发汗。下之宜大承气漫，发汗宜桂
枝泌。 (发热服次，故属脐证。和发热服康，故届表证。 )

太阳病未解，有版队阳俱停，必先振导，汗出而解。但阳且微者
先汗出而解，但险有微者，下之而解。若欲下之，宜调胃承气渴。
( 奏标则服停，表极后通，则手慑。阳服徽，腑气不实也。除服微，燥热伞津也。以上六
章，论阳明兼柴全须先汗以解表，然合可下之法。)
