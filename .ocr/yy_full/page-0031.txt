太阳病，身黄，且沉结，少腹硬，小便不利者，为悉血也。小
便自利，其人如狂，血证谤也，抵锁汤主之。 ( 桨本病时而服沉发在少腹
硬，腾胀热也。)

伤寒有热，少腹满，应小便不利，今反利者，为有血也，当下
之，不可通车，宜抵当驴。 (热不实，小便不利。必然实，小便万利。)

太阳病六七日，表证猜存，有版微而沉，反不结胸，其人发狂。
以热在下焦，少腹当硬满，小便自利者，下血帮愈。所以然者，以
太阳随经，六热在庄故也，抵当渴主之。 (某街之中，有太阳之经，脐热风
经热入庄。以上四章，论太阳膀胰腑病，则名实相符之太阳病也。太阳脏病，只有四
章。)
少阳脆经病

少阳之为病，口苦咽乾，目眩也。 (此一章，论少阳既病之提网。 )

伤寒中风五六日，寒热往来，胸居苦满，咱咯不位饮食，心烦
喜噶，或心中烦而不咀，或渴，或腹中痛，关下阁，或心下慰，小
便不利，或不渴，身有微热，或咳者，小柴胡汤主之。 (非表可计，非
宰可光可下，只可和解，故日经病。所有详请，缘少阳既所升降不和之现象。 )

血弱气画，胰理开，政气因入，和与正气相搏，结於确下。正政
分委，往来骞热，体作有时，默默不签饮食。及腑相连，其痛必
下，政高痛下，故使咀也，小柴胡渴主之。 ( 收万隆木鹿冒土，痛万肝木刻
牌土。)

伤寒四五日， 身热恶寒， 颈项强， 将下满， 手足温而淘者， 小
柴胡渴主之。 (少也经循疹下行者下满，故属少阳经病。四五日详传经篇。以上三
章，论少阳缘讶证。)

伤寒， 发热汗出不解，心下阁硬，咀而下利者，大柴胡渴主
之。 (下利用下热，辣琶乃元结，故解经兼下胃。此一章，议少阳实证，人然实在骨肺，
少阳经证仍虚也。 )
