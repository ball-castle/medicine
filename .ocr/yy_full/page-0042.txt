少阳脆经病

伤寒中风，有柴胡证，但见一证便是，不必悉具。 ( 口苦，耳莫，
目肪，咽乾，胸硬，者痛，寒热往来。)

鹃而发热者，小柴胡渴主之。 (少局脆既上送，则噶而发热。)

伤寒，阳脾湾险脾纺，法党腹中急痛者，先用小建中汤;， 不差
者，与小柴胡渴主之。 (阳渚除弦，未气散希，建中舒散，柴胡散结。主之，似多

鹃家不可与建中沁，以甜故也。 ( 菇味更缓，旺家崩逆不降，忌鞭味之过
缓。以上四章论小紫胡混用法。)

太阳病，十日已去，有浮细而嗜共者，外已解也。设胸满腹痛
者，与小柴胡沁主之。 ( 某街病通十日，嗜欧胸满，服细属少阳也。 )

伤寒六七日，发热微恶蹇，肢季烦疼，微哑，心下支结，外证
未去者，柴胡桂枝沁主之。 (微量支知，少阳证也。)

太阳与少阳合病，自下利者，与黄芬汤。若明者，黄芬加半夏
生苦沁。 (福火热而二，故少阳经熏表合病，即利。以上三章，论某衔表病与少阳经个

阳明少阳合病，必下利。其县不负者顺也，负者失也。互相克
病下利，乃经气素天之利，木克土融负，服左盛右天再负，服负态主，宿食为陪。 )

服柴胡渴已，渔者属阳上明也，以法治之。 (小混衣多热莫，阳明信
燥，故服之作渴。以上二章，论少阳与阳明合病之治法。)

妇人中风，发热恶寒，经水让来，得之七八日，热除而版肥身
其实而漓之。 ( 血内热故身泳话语。刺期门以海血热。)

刀人中风，七八日续得寒热，发作有时，经水通断者，此为热
入血室，其血必结，故使如猪状，发作有上时，小柴胡渴主之。 (三全
相火。尺服主之。血室亦尺服主之，此病尺服必动数。 )

刀人伤寒发热，经水通来之时，画日明了，莫则论语，如见鬼
状者，此为热入血室。狼犯胃和气及上二焦，则自愈。 (部入血室，幕旭
热增，故施语也。不犯胃气及上二焦，小柴胡省之法是也。以上三章论婚人经期，桌本王
