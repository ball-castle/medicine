太阳病，桂枝证，峰反下之，利遂不止。有版促者，表未解也，
喘而汗出者，葛根黄连黄芬汤主之。 (利和不止为除证，服促喘汗之利，划隐语
上。服促者名上，加一着字读，便明显。利弟不止为陪，服促嘴汗为主。 )

下和后不可更行桂枝汤，若汗出而跨无大热者，可与麻黄杏仁甘
草石语沁。 ( 污出为下家燥热，喘为肺氧实逆。交大热者，无表示之发热，身外大
热，身内即不热，即忌业方。)

发汗后不可更行桂枝汤，汗出而喘无大热者，可与麻黄杏仁甘
草石襄汤。 (和不可桂被活，言宜麻查小也，非一概不可也。 )

服桂枝汤，大汗出生，大烦渴不解，且洪大者，白虎加人参渴
主之。 (大计津小，洪大虚服，大汗又烦渴，故宣仿救津小。服洪大又渴，此洪大重按
必雪滑象也。 )

伤寒病若吐若下后，七八日不解，热结在庄，表庄俱热，时时
恶风，大渴，舌上辊煤而烦，欲饮水数升者，白虎加人参汤主之。
( 欲字作能字解，庄燥热，热主小，故恶风。庄元标，表亦熬，此表热，非表证之热。表
热重按约根，庄熟之元有根。 )

太阳病先下之而不愈，因复发汗，以此表庄俱虚，其人因致
冒，冒家汗出自愈，所以然者，汗出表和故也。得襄未和，然后下
之。 (虚万津液伤，津伪吉越故冒，津伤则屎硬。 )
