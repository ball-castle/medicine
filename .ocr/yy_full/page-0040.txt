伤寒，若吐若下和后不解，不大便五六日，上至十余日，日哺所
装潮热，不恶寒，独语如见鬼状。若剧者，发则不识人，循衣摸
床，惕而不安，微喘直视，版纺者生，版注者死，微者但发热度语
者，大承氛沁主之。若一服利，止后服。 (弦蕊木气生气，尘马无生气。得
者句，指纺物语讲诈。以上二章论也明之败请。)

发汗不解，腹满痛者，急下之，宜大承气渴。 ( 燥土借及太队之
降。)

阳明病，发热汗多者，急下之，宜大承氟沁。 (煤土伤及少险之
除。)

伤寒六七日，目中不了了，睛不和，扰表襄证，大便准，身微
热者，此为实也。急下之，宜大承气渴。 ( 燥土售及原除之除。以上三章，
治阳明非常实证。)

阳明证，其人善忘，必有畜血。所以然者，必有久凉血，故令
善忘。屎上内硬，大便反易，其色必黑，宣抵当渴下之。 ( 痛主茂乱名
氧借旭普忘，亚驴友色。 )

病人故表襄证，发热七八日，难有瞩浮数者，可下之。假令已
下，有版数不解，合热则消融善鳞，至六七日不大便者，有瘀血也，
宜抵当汤。若有瞩数不解，而下利不止，必上确热而便觅血也。 ( 浮数可
下，帮设问词。消费蔷全，血竣生风。浮数吉在经不在脐，热在经故便腑血。以上二章，
论阳明著了血之证。)

阳明病，下血诡语者，此为热入血室。但标汗出者，刺期门，
随其实而注之，溅然汗出，则您。 ( 但头出汗，时腹经热，剂期门以湾肝驹
热。此一章，论阳明病之妈人热信身室证。)

太阳病，项背强，业人歼，汗出恶风者，桂枝加葛根汤主之。 (比
装直春意，阳明经不前降，则后陷而直硬。足阳明经主前降，手也明经主后升。手阳明经
能后升，足阳明经则前降。)

太阳病，项背强，几作，故汗恶寒者，葛根渴主之。 ( 效几之项
强，灶和术葵而阳明经气亦动也，故父解之。 )

太阳与阳明合病者，必自下利，葛根渴主之。 (菜村之气，纳器慎阳
明燥玖,之氧混天。熟则氧动，吉气动则自下利。 )

太阳与阳明合病，不下利但噶者，葛根加半夏渴主之。 ( 混乱之
气盛藉下则利，盛於上则噶。)
